# Detector

Download videos and photos from pretty much any website. Works on Chrome, Firefox, Edge, Brave, and other Chromium-based browsers.

## What It Does

- Works with 150+ websites (YouTube, Instagram, Twitter/X, TikTok, Reddit, etc.)
- Handles regular video files AND streaming (HLS/M3U8)
- Pick quality when multiple options exist
- Real-time download progress
- Can handle some DRM-protected content (Widevine L3)

## Installation

### Chrome / Edge / Brave

1. Download or clone this repo
2. Open `chrome://extensions/` (or `edge://extensions/` for Edge)
3. Enable "Developer mode" (toggle in top-right)
4. Click "Load unpacked" and select the Detector folder
5. Done. You'll see the icon in your toolbar

### Firefox

1. Download or clone this repo
2. Open `about:debugging#/runtime/this-firefox`
3. Click "Load Temporary Add-on"
4. Navigate to the Detector folder and select `manifest.json`
5. Done. (Note: This is temporary and will be removed when you close Firefox. For permanent install, you'd need to sign it)

## How to Use

1. Go to any website with videos/photos
2. Click the Detector icon in your toolbar
3. It'll show you what's available to download
4. Click download

That's it. Pretty straightforward.

## Supported Websites

Over 150 sites including: YouTube, Instagram, Twitter/X, TikTok, Facebook, Reddit, Twitch, Vimeo, Dailymotion, Pornhub, Imgur, and a bunch more.

If your site isn't supported, open an issue or better yet, add it yourself and submit a PR.
